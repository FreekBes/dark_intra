FOLDER STRUCTURE:

- background: scripts used by the extension as background scripts
- popup: scripts used by the extension in the popup when you click on the extension's icon in the browser's toolbar
- shared: scripts and styling used by the Intranet in both v2 and v3
- utils: scripts used by the extension as utility scripts
- v2: scripts and styling only used in the v2 version of the Intranet
- v3: scripts and styling only used in the v3 version of the Intranet
